== Description ==

WP Colorful Tag Cloud allows you to display a nice tag cloud using different colors. 
Colors are customizable and depend on tags weights (meaning occurences) and sizes.  
A lot of other customization parameters are available.

[See plugin website description and screenshots for a full description and a live demo](http://blog.websourcing.fr/wp-colorful-tag-cloud-plugin-for-wordpress-reference-page/)


== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip and upload `wp-colorful-tag-cloud` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Cutomize your settings using the menu item under Settings>Colorful tag Cloud
1. Don't forget to save your settings
